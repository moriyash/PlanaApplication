# Plana

**Everything in one plan.**

Plana is a mobile app for shared life planning. It brings groups, category-based workspaces, activity, profile, and settings into one calm, multilingual experience, with full RTL support.

> **Repository note:** This public repository is a product preview. It is not a full open-source release of the production application.

## Overview

Plana is built for families, roommates, teams, and any group that needs one place to coordinate money, tasks, shopping, goals, and day-to-day updates without losing context.

The product is designed around a simple idea: each group gets the right workspace for its purpose, while members stay aligned through activity, invitations, and personal preferences.

## Core capabilities

- **Groups** - Create, search, archive, invite by email, and manage members.
- **Category workspaces** - Purpose-built surfaces for household, shopping, income, expenses, goals, time, projects, travel, family, personal planning, and more.
- **Activity** - A shared feed for group updates and relevant notifications.
- **Profile and settings** - Profile photo, appearance, language, notifications, privacy, and security.
- **Group identity** - Custom color, icon, and photo, plus per-member notification and archive preferences.

## Product highlights

| Area | What users get |
| --- | --- |
| Collaboration | Shared context across people and categories |
| Localization | Multilingual UI with RTL-ready layout |
| Personalization | Profile and group-level visual identity |
| Control | Notification and archive preferences per group |
